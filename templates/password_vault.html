🔒 Password Vault

เข้ารหัสด้วย AES-256-GCM ผ่าน Web Crypto API — ข้อมูลอยู่ในเบราว์เซอร์นี้เท่านั้น

ตั้งค่า Master Password ครั้งแรก

⚠️ ถ้าลืม Master Password นี้ ข้อมูลทั้งหมดใน Vault จะกู้คืนไม่ได้เด็ดขาด
เราไม่เก็บ Master Password ไว้ที่ไหนทั้งสิ้น (ไม่มีปุ่ม "ลืมรหัสผ่าน")

⚠️ ใช้งานหน้านี้ไม่ได้บนที่อยู่นี้

Password Vault ต้องใช้ Web Crypto API (crypto.subtle) ซึ่งเบราว์เซอร์อนุญาตเฉพาะบน secure context เท่านั้น คือ https:// หรือ http://localhost / http://127.0.0.1

กรุณาเปิดหน้านี้ผ่าน http://localhost:5000/password-vault หรือ http://127.0.0.1:5000/password-vault แทน (ต้องเปิดจากเครื่องที่รัน server นี้อยู่)
